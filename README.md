# SAILENDRA KHATIWADA · DIGITAL ARCHITECT PORTFOLIO

![Portfolio Preview](https://via.placeholder.com/1200x630?text=Sailendra+Khatiwada+Portfolio)

A sophisticated, modern personal portfolio website showcasing the work and skills of Sailendra Khatiwada — a Backend Developer from Kathmandu, Nepal.

## ✨ Features

### 🎨 Modern UI/UX Design
- **Elegant Dark Theme** — Premium aesthetic with purple accent colors
- **Smooth Animations** — Subtle transitions and hover effects
- **Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- **Clean Typography** — Using Playfair Display for headings and Inter for body text

### 🎮 Interactive 3D Background
- **Three.js Integration** — Rotating torus knot with particle ring
- **Subtle Visual Effects** — Enhances experience without distracting from content
- **Performance Optimized** — Lightweight 3D scene with minimal performance impact

### 📱 Complete Sections
1. **Hero Section** — Bold introduction with call-to-action buttons
2. **Skills Arsenal** — Tech stack showcase with interactive tags
3. **Projects Grid** — 6 featured projects with tech badges
4. **Resume Timeline** — Professional experience and education history
5. **About Me** — Personal background and contact information
6. **Contact Form** — Functional message submission interface

### 🔍 SEO Optimized
- Meta tags for description, keywords, and author
- Open Graph tags for social media sharing
- Twitter card support
- Structured data (JSON-LD) for rich search results
- Semantic HTML structure

### 📄 Resume Download
- One-click PDF resume download (simulated — ready for your actual PDF)
